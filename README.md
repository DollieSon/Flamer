# Flamer 🔥❤️‍🔥

Flamer is a rust based Flames game that uses your cli arguments

## How to use
### via cargo
```
cargo run "Ray Anthony" "Timothee Shamalee"
```
### via .exe
```
Flamer.exe "Timothay Chalamai" "Rey Anthony"
```

## Development
requirements:   
	rust and cargo

1. clone project.
2. run 
```
cargo run "{name1}" "{name2}"
```
